# HappyDemo

Playwright + CucumberIO + Typescript 

## Installation

### Install Repo
```bash
git clone https://github.com/mishmaccas/happydemo.git
```
### Node update
```bash
nvm install node
```
### Install
```bash
npm install
npx playwright install
```

## Usage
### Execute e2e journey
```bash
npx playwright test journey.spec.ts --project=chromium --headed
```
