# HappyDemo

Playwright + CucumberIO + Typescript 

## Installation

### Install Repo
```bash
git clone https://github.com/mishmaccas/happydemo.git
```
### Node update
```bash
nvm install node
```
### Install
```bash
npm install
npx playwright install
```

## Usage
### Execute TS spec

```bash
npx playwright test journey.spec.ts --project=chromium --headed
```

### Execute Cucumber 

```bash
npx cucumber-js --config cucumber.js
```
